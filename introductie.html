Wat en waarom?

In deze cursus gaan we aan de slag met HTML, CSS en Javascript. Technieken voor het web. Elke website die je ziet heeft HTML als basis. Ook Flash websites. HTML is wat je hier ziet. Voornamelijk tekst met codes. Je browser vertaald de codes in een vorm. En met CSS kun je die vorm mooi maken. Daarnaast kun je Javascript toevoegen voor extra interactiviteit.

Het is handig om deze basis te hebben zodat:

  • je eenvoudige websites zelf kunt maken, zowel op laptop als op tablet en mobiel
  • je beter kunt inschatten wat de mogelijkheden zijn
  • je goed kunt communiceren met web developers
  • je deze cursus kunt halen

Stel vooral je eigen doelen in deze lessen. Het maakt mij niet uit wat je wil leren. HTML5, CSS3, jQuery, gracefull degradation, Internet Explorer, Twitter bootstrap, etc. Er is alleen de eindopdracht; maak een portfoliowebsite. Overleg even als je al volledig tevreden bent met je huidige portfoliowebsite en je een andere uitdaging wil.
